HTML เบื้องต้น: หลักการและการปฏิบัติ
เรียนรู้หลักการ ทฤษฎี และการใช้งาน HTML จริง พร้อมตัวอย่างที่ชัดเจน
HTML (HyperText Markup Language) เป็น ภาษามาร์กอัป ที่ใช้สร้างเอกสารบนเว็บ HTML ไม่ใช่ภาษาโปรแกรมมิ่ง แต่เป็น ภาษาสำหรับจัดโครงสร้าง โดยใช้ Tag เป็นตัวกำหนดโครงสร้างและเนื้อหา Tag จะอยู่ในรูปแบบ <tagname> และมักจะมี ปิด Tag เป็น </tagname> เช่น <h1>หัวเรื่อง</h1> หรือ <p>ย่อหน้า</p> HTML เป็นพื้นฐานของทุกเว็บไซต์ และทำงานร่วมกับ CSS สำหรับการตกแต่ง และ JavaScript สำหรับการโต้ตอบ

โครงสร้างพื้นฐานของ HTML

HTML เป็น Tree Structure ที่มีการจัดลำดับแบบชั้นเชิง HTML ทุกไฟล์จะต้องมีโครงสร้างพื้นฐาน ดังนี้:
Tag หน้าที่ ตัวอย่าง
<!DOCTYPE html> บอกว่าเป็นไฟล์ HTML5 <!DOCTYPE html>
<html> เริ่มต้นเอกสาร HTML <html lang="th">
<head> ข้อมูลเกี่ยวกับหน้าเว็บ <head>...</head>
<title> ชื่อที่แสดงบน Tab เบราว์เซอร์ <title>หน้าแรก</title>
<body> เนื้อหาที่ผู้ใช้เห็น <body>...</body>
โครงสร้าง HTML

หลักการของ Tags และ Elements

Tag คือเครื่องหมายที่ใช้กำหนดความหมายของเนื้อหา ในขณะที่ Element คือส่วนรวมของ Opening Tag, Content และ Closing Tag หลักการสำคัญ: - Tags ต้องเปิดและปิดอย่างถูกต้อง - การซ้อน Tags ต้องเป็นไปตามลำดับชั้น - Tag Names ไม่คำนึงถึงตัวใหญ่-เล็ก แต่ควรใช้ตัวเล็กเสมอ - บาง Tags ไม่มี Closing Tag (Self-closing tags)

1. Heading Tags (หัวเรื่อง)

ใช้สำหรับสร้างหัวเรื่องในขนาดต่างๆ มี 6 ระดับ โดยมีความสำคัญทางโครงสร้างและ SEO:
Tag ตัวอย่าง ความสำคัญ
<h1> <h1>หัวเรื่องหลัก</h1> สำคัญสุด ควรมีแค่ 1 ใน 1 หน้า
<h2> <h2>หัวเรื่องรอง</h2> หัวข้อหลักของแต่ละส่วน
<h3> <h3>หัวเรื่องย่อย</h3> หัวข้อย่อยภายในส่วน
<h4> - <h6> <h4>หัวเรื่องเล็ก</h4> หัวข้อรายละเอียด

2. Paragraph Tag (ย่อหน้า)

<p> เป็น Block Element ที่ใช้สำหรับสร้างย่อหน้าข้อความ ตัวอย่าง: <p>นี่คือย่อหน้าแรก</p> การขึ้นบรรทัดใหม่: ใช้ <br> สำหรับขึ้นบรรทัดใหม่ภายในย่อหน้า หลักการ: แต่ละ <p> จะมี margin บนและล่างอัตโนมัติ

3. Image Tag (รูปภาพ)

<img> เป็น Empty Element ที่ใช้สำหรับแสดงรูปภาพ (ไม่มี closing tag) รูปแบบ: <img src="ที่อยู่รูปภาพ" alt="คำอธิบายรูป">
Attribute ความหมาย ตัวอย่าง ความจำเป็น
src ที่อยู่ของรูปภาพ src="image.jpg" จำเป็น
alt คำอธิบายรูป (สำหรับ Accessibility) alt="รูปแมว" จำเป็น
width ความกว้าง (pixels) width="300" ไม่จำเป็น
height ความสูง (pixels) height="200" ไม่จำเป็น

4. Link Tag (ลิงก์)

<a> เป็น Inline Element ที่ใช้สำหรับสร้างลิงก์ไปยังหน้าอื่นหรือเว็บไซต์อื่น รูปแบบ: <a href="ที่อยู่">ข้อความลิงก์</a>
ประเภทลิงก์ ตัวอย่าง คำอธิบาย
ลิงก์ภายในเว็บ <a href="page2.html">ไปหน้า 2</a> ลิงก์ไปยังไฟล์ในโฟลเดอร์เดียวกัน
ลิงก์เว็บไซต์อื่น <a href="https://google.com">Google</a> ต้องมี http:// หรือ https://
เปิดหน้าใหม่ <a href="https://google.com" target="_blank">Google</a> target="_blank" เปิดในแท็บใหม่
ลิงก์อีเมล <a href="mailto:test@email.com">ส่งอีเมล</a> เปิดโปรแกรมอีเมลโดยอัตโนมัติ

Elements และ Attributes

Elements ประกอบด้วย 3 ส่วน: Opening Tag, Content และ Closing Tag Attributes คือคุณสมบัติเพิ่มเติมที่อยู่ใน Opening Tag
ประเภท Element คำอธิบาย ตัวอย่าง
Block Elements ครองพื้นที่เต็มความกว้าง และขึ้นบรรทัดใหม่ <div>, <p>, <h1>
Inline Elements ครองพื้นที่เฉพาะเนื้อหา อยู่ในบรรทัดเดียวกันได้ <span>, <a>, <strong>
Empty Elements ไม่มี Closing Tag และไม่มีเนื้อหาข้างใน <img>, <br>, <hr>
Global Attributes ใช้ได้กับ Tag ทุกประเภท id, class, style, title

หลักไวยากรณ์ HTML

กฎการเขียน HTML: - ต้องเริ่มด้วย <!DOCTYPE html> - Tag Names เขียนด้วยตัวเล็ก - Attribute Values ต้องอยู่ในเครื่องหมายคำพูด - การซ้อน Tags ต้องเป็นไปตามลำดับ (LIFO - Last In First Out) - เว้นวรรคและขึ้นบรรทัดใหม่เพื่อให้อ่านง่าย

ขั้นตอนการสร้างไฟล์ HTML แรก

  1. เปิดโปรแกรม Visual Studio Code
  2. สร้างไฟล์ใหม่ ตั้งชื่อ index.html
  3. พิมพ์โครงสร้างพื้นฐาน HTML เช่น <!DOCTYPE html>, <html>, <head>, <body>
  4. เพิ่มเนื้อหาใน <body> เช่น ข้อความ, รูปภาพ, ลิงก์
  5. บันทึกไฟล์ แล้วเปิดด้วย Web Browser
  6. ตรวจสอบผลลัพธ์ และแก้ไขตามต้องการ
