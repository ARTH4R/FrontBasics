Flexbox และ Grid Layout
เรียนรู้การจัด Layout ด้วย Flexbox และ CSS Grid สองเครื่องมือที่ทรงพลังสำหรับการออกแบบเว็บไซต์สมัยใหม่
Flexbox และ CSS Grid คือเครื่องมือสำคัญสำหรับการจัด Layout ในเว็บไซต์สมัยใหม่ Flexbox เหมาะสำหรับ การจัดเรียงแบบ 1 มิติ (แนวนอนหรือแนวตั้ง) CSS Grid เหมาะสำหรับ การจัดเรียงแบบ 2 มิติ (ทั้งแนวนอนและแนวตั้งพร้อมกัน) การเข้าใจทั้งสองระบบทำให้เราสามารถ สร้าง Layout ที่ซับซ้อนได้อย่างง่ายดาย และ Responsive

Flexbox พื้นฐาน - Flexible Box Layout

Flexbox คือระบบ Layout แบบ 1 มิติ ที่ช่วยจัดเรียง items ในแนวนอนหรือแนวตั้ง เริ่มต้นใช้งาน: display: flex; แนวคิด: มี Container (parent) และ Items (children)
CSS Code คำอธิบาย ผลลัพธ์
.container { display: flex; } เปิดใช้งาน Flexbox Items จะเรียงในแนวนอน
flex-direction: row; ทิศทางแนวนอน (ค่าเริ่มต้น) ← → (ซ้ายไปขวา)
flex-direction: column; ทิศทางแนวตั้ง ↑ ↓ (บนลงล่าง)
flex-direction: row-reverse; แนวนอนกลับทิศ → ← (ขวาไปซ้าย)
ข้อดีของ Flexbox: - จัดเรียง items ง่ายและรวดเร็ว - ควบคุมการกระจายพื้นที่ได้อย่างยืดหยุ่น - จัดตำแหน่งแนวนอนและแนวตั้งได้สะดวก - เหมาะสำหรับ Navigation, Cards, Buttons

Flex Container Properties

Properties สำหรับ Container (parent element)
Property ค่าที่ใช้ได้ คำอธิบาย ตัวอย่าง
justify-content flex-start, center, flex-end, space-between, space-around จัดตำแหน่งแนว Main Axis จัดให้อยู่ตรงกลาง
align-items flex-start, center, flex-end, stretch, baseline จัดตำแหน่งแนว Cross Axis จัดให้อยู่บนหรือล่าง
flex-wrap nowrap, wrap, wrap-reverse กำหนดว่าจะขึ้นบรรทัดใหม่หรือไม่ wrap เมื่อพื้นที่ไม่พอ
gap px, rem, % ระยะห่างระหว่าง items gap: 1rem;

Flex Item Properties

Properties สำหรับ Items (child elements)
Property คำอธิบาย ตัวอย่าง
flex-grow ขยายตัวเมื่อมีพื้นที่เหลือ flex-grow: 1; (ขยายเท่ากัน)
flex-shrink หดตัวเมื่อพื้นที่ไม่พอ flex-shrink: 0; (ไม่หด)
flex-basis ขนาดเริ่มต้น flex-basis: 200px;
flex shorthand ของ grow shrink basis flex: 1 0 auto;
align-self จัดตำแหน่งของ item นี้เอง align-self: center;

CSS Grid Layout พื้นฐาน

CSS Grid คือระบบ Layout แบบ 2 มิติ ที่สามารถจัดการทั้งแถวและคอลัมน์พร้อมกัน เริ่มต้นใช้งาน: display: grid; แนวคิด: แบ่งพื้นที่เป็นตาราง (grid) แล้ววาง items ลงไป
Property คำอธิบาย ตัวอย่าง
grid-template-columns กำหนดจำนวนและขนาดคอลัมน์ grid-template-columns: 200px 1fr 1fr;
grid-template-rows กำหนดจำนวนและขนาดแถว grid-template-rows: 100px auto;
gap / grid-gap ระยะห่างระหว่าง cells gap: 20px; (แถวและคอลัมน์)
grid-column / grid-row กำหนดตำแหน่ง item grid-column: 1 / 3; (ครอบ 2 คอลัมน์)
หน่วยพิเศษใน Grid: - fr (fraction) - แบ่งพื้นที่ตามสัดส่วน - auto - ขนาดตามเนื้อหา - repeat() - ทำซ้ำ เช่น repeat(3, 1fr) - minmax() - กำหนดขนาดต่ำสุดและสูงสุด

Flexbox ขั้นสูง - Alignment และ Distribution

Technique CSS Code การใช้งาน
Center ทุกอย่าง display: flex; justify-content: center; align-items: center; จัดกึ่งกลางทั้งแนวนอนและตั้ง
Space Between justify-content: space-between; กระจาย items ห่างเท่าๆ กัน
Flexible Cards flex: 1 1 300px; (grow shrink basis) Cards ขยายหดได้ ขนาดเริ่มต้น 300px
Sticky Footer body: flex column; footer: margin-top: auto; Footer อยู่ล่างสุดเสมอ
Equal Height Columns align-items: stretch; (default) คอลัมน์สูงเท่ากันอัตโนมัติ

Grid Template Areas - การวาง Layout แบบภาพ

Template Areas ช่วยให้เราวาง Layout ได้อย่างมองเห็นภาพชัดเจน
CSS Code คำอธิบาย
grid-template-areas:
"header header header"
"sidebar main main"
"footer footer footer"; 		วาง Layout แบบ 3 คอลัมน์
Header ครอบทั้งหมด
Sidebar 1 คอลัมน์, Main 2 คอลัมน์
Footer ครอบทั้งหมด
.header { grid-area: header; } กำหนดให้ element นี้อยู่ใน area ชื่อ header
grid-template-columns: 200px 1fr 1fr; กำหนดขนาดคอลัมน์ที่สอดคล้องกับ areas
Grid Auto Placement: - grid-auto-flow: row (ค่าเริ่มต้น) หรือ column - grid-auto-rows: ขนาดแถวที่สร้างอัตโนมัติ - grid-auto-columns: ขนาดคอลัมน์ที่สร้างอัตโนมัติ

Responsive Design ด้วย Flexbox และ Grid

เทคนิค Flexbox Grid
Auto Responsive Cards flex-wrap: wrap; flex: 1 1 300px; grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
เปลี่ยนทิศทางบนมือถือ @media: flex-direction: column; @media: grid-template-columns: 1fr;
ซ่อน/แสดง Items order: -1; (เปลี่ยนลำดับ) grid-column: 1 / -1; (ครอบทุกคอลัมน์)
auto-fit vs auto-fill: - auto-fit: collapse empty tracks (เหมาะเมื่อต้องการให้ items ขยายเต็ม) - auto-fill: keep empty tracks (เหมาะเมื่อต้องการขนาดคงที่)

เปรียบเทียบ Flexbox vs Grid

ลักษณะ Flexbox CSS Grid
มิติ 1 มิติ (แนวเดียว) 2 มิติ (แนวนอนและตั้ง)
เหมาะสำหรับ Navigation, Buttons, Small Components Page Layout, Complex Grids, Card Layouts
ควบคุม Layout Items ควบคุมขนาดตัวเอง Container ควบคุมทุกอย่าง
ความยืดหยุ่น ยืดหยุ่นสูง ปรับตามเนื้อหา โครงสร้างชัดเจน แม่นยำ
Browser Support รองรับเก่ากว่า (2012) รองรับใหม่กว่า (2017)
ใช้ร่วมกันได้หรือไม่ ใช้ร่วมกันได้! Grid สำหรับ Layout หลัก, Flex สำหรับ Components

Layout Patterns ที่ใช้บ่อย

Pattern วิธีทำ (Flexbox) วิธีทำ (Grid)
Holy Grail Layout Flex column + Flex row nested grid-template-areas ง่ายกว่า
Card Grid flex-wrap: wrap; repeat(auto-fit, minmax());
Centered Box justify-content + align-items place-items: center;
Sidebar Layout Sidebar fixed width, Main flex: 1 grid-template-columns: 250px 1fr;
12-Column Grid ซับซ้อน ใช้ % หรือ calc() repeat(12, 1fr); ง่ายมาก

ตัวอย่างการใช้งานจริง

1. Navigation Bar (Flexbox) .nav {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 1rem 2rem;
} 2. Card Grid (CSS Grid) .card-container {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(300px, 1fr));
  gap: 2rem;
} 3. Dashboard Layout (Grid + Flex) .dashboard {
  display: grid;
  grid-template-areas:
    "sidebar header header"
    "sidebar main main";
  grid-template-columns: 250px 1fr 1fr;
}
.widget {
  display: flex;
  flex-direction: column;
}
Use Case แนะนำใช้ เหตุผล
Navigation Menu Flexbox เรียงในแนวเดียว ยืดหยุ่น
Photo Gallery Grid จัดเป็นตาราง ควบคุมง่าย
Form Layout Grid จัด label และ input เป็นคอลัมน์
Button Group Flexbox จัดเรียงปุ่มติดกัน
Blog Layout Grid (main) + Flex (cards) ใช้ร่วมกัน

เคล็ดลับและข้อควรระวัง

Flexbox Tips: - ใช้ gap แทน margin ระหว่าง items (ทันสมัยกว่า) - flex: 1 คือ shorthand ของ flex: 1 1 0 - min-width: 0 ป้องกัน items ล้นออกมา - ระวัง flex-shrink อาจทำให้รูปบิดเบี้ยว (ใช้ 0 ถ้าไม่ต้องการให้หด) Grid Tips: - ใช้ fr แทน % เพื่อคำนวณง่าย - minmax(300px, 1fr) สร้าง responsive grid อัตโนมัติ - ใช้ grid-auto-flow: dense เติมช่องว่างอัตโนมัติ - ระวัง implicit grid (แถว/คอลัมน์ที่สร้างเอง) อาจมีขนาดไม่ตรงต้องการ ข้อควรระวังทั่วไป: - ไม่ควรใช้ float กับ Flexbox/Grid (จะไม่ทำงาน) - Margin auto ใน Flexbox ใช้จัด alignment ได้ - Grid item ที่มี position: absolute จะออกจาก grid flow

Best Practices

แนวทางปฏิบัติที่ดี: - Mobile First: เริ่มจาก 1 คอลัมน์ แล้วค่อยขยาย - ใช้ร่วมกัน: Grid สำหรับ page layout, Flex สำหรับ components - Semantic HTML: ใช้ tags ที่เหมาะสม ไม่ใช้ div ทั้งหมด - Comment: อธิบาย layout ที่ซับซ้อน - Test: ทดสอบบน browsers และขนาดหน้าจอต่างๆ
❌ ไม่ควรทำ ✅ ควรทำ
ใช้ Flexbox สำหรับทั้ง page layout ใช้ Grid สำหรับ page layout
ใช้ fixed width ทุก element ใช้ fr, %, auto ให้เหมาะสม
ซ้อน container หลายชั้นโดยไม่จำเป็น ใช้ Grid nested เมื่อจำเป็น
ใช้ !important บังคับ layout เข้าใจ specificity และแก้ที่ต้นเหตุ
Performance: - Flexbox และ Grid มี performance ดีมาก ไม่ต้องกังวล - หลีกเลี่ยง deep nesting (ซ้อนหลายชั้นเกินไป) - ใช้ CSS variables สำหรับค่าที่ซ้ำ (เช่น gap, padding)
