CSS พื้นฐาน: การตกแต่งเว็บไซต์
เรียนรู้หลักการ ทฤษฎี และการใช้งาน CSS เพื่อสร้างเว็บไซต์ที่สวยงาม
CSS (Cascading Style Sheets) คือ ภาษาสำหรับตกแต่งเว็บไซต์ ที่ทำงานร่วมกับ HTML หาก HTML เป็น โครงสร้าง ของเว็บไซต์ CSS ก็เป็น การตกแต่ง ที่ทำให้เว็บไซต์ดูสวยงามและใช้งานง่าย CSS ช่วยกำหนด สี ฟอนต์ ขนาด ตำแหน่ง และเลย์เอาท์ ของ HTML Elements หลักการสำคัญของ CSS คือการ แยกเนื้อหาออกจากการตกแต่ง ทำให้จัดการและบำรุงรักษาง่าย

หลักไวยากรณ์ CSS

CSS ประกอบด้วย Selector และ Declaration Block ที่มี Properties และ Values
ส่วนประกอบ คำอธิบาย ตัวอย่าง
Selector เลือก HTML Element ที่ต้องการตกแต่ง h1, .class, #id
Property คุณสมบัติที่ต้องการเปลี่ยน color, font-size, margin
Value ค่าของ Property red, 16px, 10px
Declaration Property + Value color: red;
รูปแบบ: selector { property: value; } ตัวอย่าง: h1 { color: blue; font-size: 24px; }

3 วิธีการเขียน CSS

วิธีการ คำอธิบาย ตัวอย่าง ข้อดี/ข้อเสีย
Inline CSS เขียนใน HTML Tag โดยตรง <h1 style="color: red;">หัวเรื่อง</h1> เร็ว แต่ยากจัดการ
Internal CSS เขียนใน <style> ในส่วน <head> <style> h1 { color: red; } </style> เฉพาะหน้า ใช้ได้กับหน้าเดียว
External CSS เขียนในไฟล์ .css แยกต่างหาก <link rel="stylesheet" href="style.css"> ใช้ซ้ำได้ จัดการง่าย (แนะนำ)

CSS Selectors (ตัวเลือก)

Selectors คือวิธีการเลือก HTML Elements ที่ต้องการตกแต่ง
ประเภท Selector รูปแบบ ตัวอย่าง คำอธิบาย
Element Selector tagname h1 { color: red; } เลือกทุก tag ประเภทนั้น
Class Selector .classname .highlight { background: yellow; } เลือกตาม class attribute
ID Selector #idname #header { font-size: 30px; } เลือกตาม id attribute (ใช้แค่ครั้งเดียว)
Universal Selector * * { margin: 0; } เลือกทุก element

Cascading และ Inheritance

Cascading หมายถึงการที่ CSS Rules มีลำดับความสำคัญ และสามารถทับซ้อนกันได้ ลำดับความสำคัญ (จากมากไปน้อย): 1. Inline styles (style="...") 2. IDs (#id) 3. Classes (.class), attributes, pseudo-classes 4. Elements (h1, p, div) Inheritance: บาง Properties สืบทอดจาก Parent Element เช่น font-family, color ไม่สืบทอด: Properties เกี่ยวกับ layout เช่น margin, padding, border

Specificity (ความจำเพาะ)

เมื่อมี CSS Rules หลายตัวชี้ไปยัง Element เดียวกัน Browser จะใช้ Rule ที่มี Specificity สูงสุด
Selector Specificity Score ตัวอย่าง
Inline style 1000 style="color: red;"
ID 100 #header
Class 10 .highlight
Element 1 h1

CSS Box Model

ทุก HTML Element ถือว่าเป็น "กล่อง" ที่ประกอบด้วย 4 ส่วน: 1. Content: เนื้อหาจริง (ข้อความ, รูปภาพ) 2. Padding: พื้นที่ว่างรอบเนื้อหา (ภายในกล่อง) 3. Border: เส้นขอบของกล่อง 4. Margin: พื้นที่ว่างรอบกล่อง (ภายนอกกล่อง) ขนาดรวม = Content + Padding + Border (Margin ไม่นับ)

Text Properties

Property คำอธิบาย ตัวอย่าง Values
font-family ประเภทฟอนต์ "Arial", "Kanit", sans-serif
font-size ขนาดฟอนต์ 16px, 1.2em, 120%
font-weight ความหนาของฟอนต์ normal, bold, 400, 700
text-align การจัดตำแหน่งข้อความ left, center, right, justify
line-height ความสูงของบรรทัด 1.5, 24px, 150%
text-decoration การตกแต่งข้อความ none, underline, line-through

Color Properties

Property คำอธิบาย ตัวอย่าง Values
color สีข้อความ red, #FF0000, rgb(255,0,0)
background-color สีพื้นหลัง blue, #0000FF, rgba(0,0,255,0.5)
background-image รูปภาพพื้นหลัง url("image.jpg")
opacity ความโปร่งใส 0.5, 0.8 (0 = โปร่งใส, 1 = ทึบ)
รูปแบบสี: - ชื่อสี: red, blue, green - Hex: #FF0000 (แดง), #00FF00 (เขียว) - RGB: rgb(255, 0, 0) (แดง) - RGBA: rgba(255, 0, 0, 0.5) (แดงโปร่งใส 50%)

Layout Properties

Property คำอธิบาย ตัวอย่าง Values
display ลักษณะการแสดงผลของ Element block, inline, none, flex
width ความกว้าง 300px, 50%, auto
height ความสูง 200px, 50vh, auto
position การจัดตำแหน่ง static, relative, absolute, fixed
float การลอยของ Element left, right, none

Spacing Properties (Box Model)

Property คำอธิบาย ตัวอย่าง
margin พื้นที่ว่างภายนอก Element 10px, 10px 20px, auto
padding พื้นที่ว่างภายใน Element 15px, 10px 20px 15px 5px
border เส้นขอบ 1px solid black
border-radius มุมโค้ง 5px, 10px 20px
การเขียนแบบย่อ: - 1 ค่า: margin: 10px; (ทุกด้าน) - 2 ค่า: margin: 10px 20px; (บน-ล่าง ซ้าย-ขวา) - 4 ค่า: margin: 10px 20px 15px 5px; (บน ขวา ล่าง ซ้าย)
